При виставленні швидкості 1200 по uart модуль може не відкликатися. тому треба слати йому команди через термінал.
дані потрібно слати у такому вигляді 
0x5A 0x68 0x04 0x57 0x50 СS 0xA5
CS=(0x5A+0x68+ Register-1+ Register-2+ Register-3)%256
з пробілами. CS - контрольна сума чи щось типу того. дані тре слати не в ASCII коді, а просто цифрами!

0x5A 0x68 0x03 0x37 0x32 0x2E 0xA5
